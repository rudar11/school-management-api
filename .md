# School Management API

A Node.js & MySQL based API to manage schools and find the nearest ones based on user location.

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Libraries:** geolib (for distance calculation), mysql2, dotenv

## API Endpoints

### 1. Add School
- **URL:** `/api/addSchool`
- **Method:** `POST`
- **Payload:**
  ```json
  {
    "name": "Springdales School",
    "address": "Delhi",
    "latitude": 28.5915,
    "longitude": 77.1685
  }
  
  ### 2. List Schools
- **URL:** `/api/listSchools`
- **Method:** `GET`
- **Query Params:** `latitude`, `longitude` (User's current location)
- **Response:** A JSON array of schools sorted by proximity (nearest first), including a `distance_km` field.
- **Example:** `/api/listSchools?latitude=28.6139&longitude=77.2090`
